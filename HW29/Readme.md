## 1. Зарегистрироваться на облачном провайдере Google Cloud Platform (GCP)
```
- В качестве локации указать Грузию.
- Привязать банковскую карту РБ/РФ.
- Получить кредиты от GCP (300$) путем верификации карты.
```
